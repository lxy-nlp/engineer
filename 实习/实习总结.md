## 飞单项目

1. 背景

   司机在平台上得到拉货信息,司机与货主联系后, 没有在平台成交,而是私下成交, 飞单检测的目的是通过二人通话记录(转文本),判定是否私下成交,平台将对其采取惩罚措施.

2. 做法

   找到货主下架货源前最后一通电话的联系的司机,将和这个司机通话记录转换成文本(下架之前),对其进行打标

   1. SVM

      将文字根据词频转换为向量,放入SVM中训练

   2. 规则

      根据文本中是否出现一些关键字,编写一些规则语句,主要使用正则表达式进行. 

3. 结果

   SVM的方法可以达到68%的准确率

   规则的方法可以达到75%

4. 困难和总结

   多人打标不一致
   
5.  流程

   - 根据下架的货主id找到最后一通电话的司机id，并将之前所有和这个司机的通话全部拉下来
   - 数据预处理 繁体转换 数字转换 停用词处理
   - 构造svm特征 基于词频统计 如果超过设定阈值1那么就加到词表里；把句子里的单词转换如果在词表中出现那么就将其权重设为1
   - 处理后的数据 label callid featuresmap 
   - libsvm训练模型 grid.py 寻优

## 成交时间预测

货源成交时间预测

1. 背景 

   货主发布货源时,给出一个货源成交时间,类似于滴滴打车时预计多少分钟司机会接单.

2. 做法

   从各个表中拉取相关特征,筛选并构造特征,放入xgboost中进行训练

   取特征的依据：

   - 根据自己对业务的理解 数据表中拉数据 

   -  根据原有模型 特征重要性排序的结果 优先找重要性高的特征及其相关的特征。

   扩充特征后再去训练

   加入统计特征

   - 7日 统计 30日统计

   - 货主在这条线路上的成交率

   - 车长 车型 等车辆特征的成交率

   XGB的代码详解

   ```python
   归一化和标准化
   归一化是将数据变成0-1之间的小数
   标准化是将数据变成 方差为1 均值为0的小数
   ```

   

3. 结果

   召回相较于之前的版本提高10-20%

   精准0-5min有下降,其余均有提高

## 飞行器四维轨迹预测

1. 背景

   民用航空飞机的四维轨迹预测, 实时预测飞机的经度纬度 高度 时间

   原有的算法没有考虑高空风的影响,预测精准度没达到预期

2. 工作内容

   算法是建立在物理模型上的,主要是利用

   负责高空风算法研究, 将高空风 这个特征加入 现有的算法中 

   主要是在查找相关论文, 将其中的三角形合成向量思想应用在现有模型中将风速与飞机真空速合成

   ![image-20210608094011158](/home/lxy/文档/学习笔记/img/image-20210608094011158.png)

   ```c
   double Grb_AmendTasByGrib(double InTas, int InDirect, STU_FDP_4DMetar *InMetar)
   {
   	double d_v;
   	double d_da, d_wa, d_tmp;
   
   	d_wa = fabs(InMetar->drct_wind - InDirect);
   	if (0 == d_wa || 0 == fabs(d_wa - 180)) 
   	{
   		d_v = InTas + cos(d_wa * FDPPI / 180) * InMetar->v_wind;
   		return d_v;
   	}
   	else if (180 < d_wa)
   	{
   		d_wa = 360 - d_wa;
   	}
   
   	d_wa *= (FDPPI / 180);
   	d_da = asin(sin(d_wa) * InMetar->v_wind / InTas);
   
   	d_tmp = sin(d_wa) / InTas;
   	d_v = sin(FDPPI - d_wa - d_da) / d_tmp;
   
   	return d_v;
   }
   
   double Grb_CalTasEliminateGrib(double InSpeed, int InDirect, STU_FDP_4DMetar *InMetar)
   {
   	double d_v;
   	double d_wa;
   
   	d_wa = fabs(InMetar->drct_wind - InDirect);
   	if (0 == d_wa || 0 == fabs(d_wa - 180))
   	{
   		d_v = InSpeed - cos(d_wa * FDPPI / 180) * InMetar->v_wind;
   		return d_v;
   	}
   	else if (180 < d_wa)
   	{
   		d_wa = 360 - d_wa;
   	}
   
   	d_wa *= (FDPPI / 180);
   	d_v = sqrt(pow(InSpeed, 2) + pow(InMetar->v_wind, 2) - 2 * InSpeed * InMetar->v_wind * cos(d_wa));
   
   	return d_v;
   }
   
   ```

   

3. 本质是正余弦定理的使用

   将正余弦定理应用到矢量三角形 合成中 

   高空风和真空速是360度的 所以在应用余弦定理时要进行范围的转换

   另外对于平角的情况 0度 和180度要分开讨论

4. 结果

   到走廊口 的误差在5%以内
   
5. 总结

   从论文去找到适合问题的解决方法

   并且对于问题的一些特殊情况 对算法进行改进
